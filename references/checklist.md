# AI味分层特征清单（诊断与手术专用）

> ⚠️ 白熊警告：本清单全是坏例子。只在诊断和结构手术时加载；成稿阶段放下它，回到SKILL.md的「你相信什么」。
> 权重排序：经验层 ≥ 结构层 > 句式层 > 词汇层。词汇层最显眼也最不重要——它可以被低成本消除，消除后骗得过机器检测器，骗不过熟练读者。

## 一、经验层（最深，唯一AI伪造不了的层）

| 特征 | 识别方法 | 处方 |
|------|---------|------|
| 可替换的具体 | 对每个细节自问「换成别的例子还成立吗」。「前几天吃饭，菜还没上」这种干净得像广告分镜的场景（没有饭馆名、没有多余细节）是典型 | 换成可溯源的一手素材，或[待补] |
| 两头堵的判断 | 「方向是对的，但还需要观察」「我倾向于A，但B也有道理」——立场永远安全 | 改成可证伪的押注（内容须来自作者真实立场） |
| 无代价的情绪 | 「我震惊了」「让我后背发凉」「读到这里我坐直了」——身体反应式标注，替读者演示该有什么感受 | 删标注，让情绪从事实堆叠里溢出（先列七个会议名，再来一句「要吐了」） |
| 作者不在场 | 全文找不到一个只有作者本人写得出的句子 | 挖素材注入，见SKILL.md第一步 |

## 二、结构层（真人读者识别AI的主要线索）

| 特征 | 识别方法 | 处方 |
|------|---------|------|
| 金句锤 | 每个小节末尾落一记短句金句（对仗、可截图），位置可预测——读者到倒数第二段就知道锤要来 | 全文金句≤2处且不落节末；真人金句是随手甩的残句，不是砸实的对仗 |
| 导游路标 | 「得从头讲」「记住这个X」「后面细说」「先说A，不然B的分量出不来」「把这两件事合起来看」 | 删除，段落直接并置，读者自己接线 |
| 完美闭环 | 结尾回扣开头/标题、升华收束、双保险式结尾（升华完再补一句） | 砍到最后一个有信息量的事实或判断为止。真人写到没话说就停 |
| 段落化免责 | 「这一段是我的推测，我们到时候回来验证」「先交代一下成色」——像论文limitations小节的仪式化hedge | 拆散内嵌成口语hedge：「估计」「我猜」「不过这个我还没验证过」 |
| 匀速感 | 段落长度均匀、句句20字上下、每段恰好一个论点 | 变速：短句敢单独成段，长论证敢一口气不换气。但别机械长短交替，那是新套路 |
| 挑战与展望公式 | 「尽管…仍面临挑战…未来可期」式收尾 | 删 |

**不是病灶，别误杀**：小标题、照顾外行的解释、重点前置。盲测实证外行读者的完读靠它们承重。

## 三、句式层

| 特征 | 例子 | 处方 |
|------|------|------|
| 否定排比全家族 | 「不是X，是Y」「不是X，而是Y」「这不是A，是B」「我不是在C，我是在D」，**不论分句长短**（长变体最容易漏网） | 直接陈述Y。描述存在的东西，不描述不存在的东西 |
| 三段工整列举 | 「A卷a，B卷b，C卷c」「不用装，不用扫，不用配」 | 拆成不对称表达 |
| 工整对偶 | 「在X上是a；在Y上是b」「水下的部分可以被测量，水上的部分可以被读取」 | 打破对称，允许一边长一边短 |
| 系动词回避 | 「充当着」「扮演着」「标志着」「彰显了」替代「是」 | 换回「是」 |
| 句尾分词式浮浅分析 | 中文表现为「……，体现了/凸显了/推动了X」挂在句尾但不给证据 | 给证据，或删 |
| 破折号滥用 | 真人中文写作平均每篇不到1个 | 全文≤2处 |
| 引号给概念套壳 | 「深度工作」「第二大脑」式——概念、转述、自造词都裹上「」 | 引号只留他人原话和直接引语；逐对质询「这是谁的原话？」 |

## 四、词汇层（及格线，别止步于此）

- 衔接套话：值得注意的是、需要指出的是、首先/其次/最后、综上所述、总而言之、不难看出、由此可见
- 空洞大词：赋能、闭环、抓手、底层逻辑、深度剖析、旨在、生态（无动作托住的抽象名词）
- 万能动词：推进、打造、建构、实现（删掉一半意思不损失）
- 营销腔：显著提升、极具价值、令人惊叹、未来可期
- 导语与推销收尾：「今天我们来聊聊……」「还等什么？赶紧试试吧！」

## 使用纪律

1. **看特征簇，不看孤例**：单个破折号、一句对偶不是证据；同层特征密集共现才是
2. 专业流畅、语法正确不是AI证据，别把「写得好」当「AI写的」
3. 词表随模型版本漂移，本清单的词汇层需定期更新；结构层和经验层特征稳定得多
